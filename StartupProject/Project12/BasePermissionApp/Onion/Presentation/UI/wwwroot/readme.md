# WWWROOT
## Klasörler
**Klasör isimlerini tekil olarak ele alıyoruz çünkü düzensiz ingilizce isimler hiyerarşiyi bozuyor.**
```
	main: /Home/Index sourcelarının olduğu sayfanın adıdır.
	cork: Admin paneli ve Cork teması için kullanılan sourceları ele alır.
	asset: Varsayılan genel iskelet için gerekli dosyaların tutulduğu yapıdır.
	data: Fazladan asset/vendor harici dosya tipleri burada tutulacaktır.
```
